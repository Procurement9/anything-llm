## Filing Instructions

- File original + 2 copies with: Clerk of Court, Bamberg County
- Address: Bamberg County Courthouse, 101 Main Street, Bamberg, SC 29003
- Phone: (803) 245-3025
- Serve one copy on: Office of the Solicitor, Second Judicial Circuit
- Keep one copy for your records
- If a bench warrant is active, file Motion to Quash FIRST before any other motions
